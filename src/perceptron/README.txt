How to run perceptron.py

1. Go to where file is located: cd src/perceptron
2. Run file: python3 perceptron.py ionosphere.data

How to run perceptron_split.py

1. Go to where file is located: cd src/perceptron
2. Run file: python3 perceptron_split.py ionosphere.data

BOTH FILES ARE USING RANDOMISED WEIGHTS
- On file perceptron.py comment line 25 and uncomment line 24 to run with weights at zeros (vice-versa)
- On file perceptron_bias.py comment line 24 and uncomment line 23 to run with weights at zeros (vice-versa)
